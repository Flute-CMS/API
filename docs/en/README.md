# Flute CMS API Documentation

Welcome to the API documentation for Flute CMS.

Use the sidebar to navigate through the available endpoints. Each section corresponds to a specific resource and details the available operations, required parameters, and example responses.

## Authentication

All requests to the API must be authenticated using an API key. The key must be sent in the `X-API-KEY` header.

`X-API-KEY: <YOUR_API_KEY>`
